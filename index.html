SUMMARY OF HTML LESSON

HTML: an acronym for HyperText MarkUp Language is the language of the web. It is the basics of the web development to bring to life web apps and web pages, using texts, images, audio, video etc.

ADDING AN IMAGE

HTML consists of 3 important parts namely:

TAGS: Tags are used to describe the content of a web page. It consists of character that are enclosed in an angle brackets. Tags are of 2 forms; the opening and closing tags and a self enclosing tag. The closing tags are differentiated from the opening tags with an addition of a forward slash in the angle bracket while the self enclosing tags do not have a closing tags attached to them. Opening and closing tags are represented by <p> and </p>

ATTRIBUTES: This gives additional information or meaning to a tag. Examples of tags with attributes are image tag, input tag, audio, video etc. In the image tag for instance, <img src="" alt=""> the src and alt are attributes. The src specifies the source or path where the image is located or found. without the src the image cannot be displaced on a web page.
ELEMENTS: This refers to both the tags and content displayed on a webpage i.e <opening tag> contents <closing tag>

HTML FORMATTER

These are tags that are used to make texts appear in a certain way aside from the usual. It is done to either show importance or to stress out a point.
Examples of such tags are:

  • bold: To make a text bold
  • em: To make text stand out
  • strong: To show importance
  • i: style to make texts lean to the right
  • del: to strike out text
  • ins: defines text that has been inserted to a document
  • mark: used to highlight for reference
  • sup: 12th
  • sub: H2O
  • pre: 
    To define preformatted text
  • small: To make a text smaller
  • hr:
    To make an horizontal line
